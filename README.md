##### 编译源码

> 1. 下载Go语言安装包并安装
>
>    https://go.dev/（官网）
>
>    https://studygolang.com/dl（镜像站）
>
> 2. 配置Go语言环境
>
>    * cmd执行命令
>
>      ```bash
>      go env -w GO111MODULE=on
>      go env -w GOPROXY=https://goproxy.cn,direct
>      ```
>    
> 3. 编译
>
>    *  打开cmd进入源码目录（也可以直接在源码所在目录的地址栏，输入cmd）
>
>    * cmd执行命令
>
>      ```bash
>      go build main.go
>      ```
>
>
